# SimpleJQuery

Simple jQuery mock to be used in console tests, WIP

## Installation

Via npm on Node:

```
npm install simplejquery
```

## Usage

Reference in your program:

```javascript
var sjq = require('simplejquery');
var $ = sjq.$;
var $ = sjq.$$;
```
